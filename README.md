# Getting Started with React DEMO

Demo React [GitHub Pages](https://urbanmarcen.github.io/)

## Short Info

The project is based on:
- React Create App
- Material UI
- Redux - global state management
- React Context API - local state management for isolated components
- i18n - translation management
- React Router
- Axios

Authentication based on JWT tokens with 15minutes expiration time

In the project directory, you can run:

### `npm test`

### `npm run build`
Run production build

### `npm run deploy`
Run deploy to github-pages

### `npm run eject`


### `npm run translate`
